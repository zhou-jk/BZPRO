#1043. [HAOI2008]下落的圆盘

内存限制：162 MiB 时间限制：10 Sec
提交 提交记录 讨论

题目描述

　　有n个圆盘从天而降，后面落下的可以盖住前面的。求最后形成的封闭区域的周长。看下面这副图, 所有的红
色线条的总长度即为所求.

输入格式

　　第一行为1个整数n,N<=1000
接下来n行每行3个实数,ri,xi,yi,表示下落时第i个圆盘的半径和圆心坐标.

输出格式

　　最后的周长，保留三位小数

样例

样例输入


			
2

1 0 0

1 1 0

样例输出


			
10.472
