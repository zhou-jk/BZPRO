#4324. tjoi2012 炸弹

内存限制：512 MiB 时间限制：10 Sec
提交 提交记录 讨论

题目描述

在平面上有n个炸弹[1..n]，每个炸弹的爆炸范围是|x-xi|+|y-yi|<=R，如果某个炸弹在爆炸了，那么它将引燃它
范围内的所有炸弹。现要求出至少引燃多少炸弹才能使得所有炸弹都爆炸。

输入格式

第一行两个整数n,r
接下来n行，每行两个整数xi,yi，炸弹的坐标。
1<=n<=100000，0<=r<=10^9, 0<=xi,yi<=10^9

输出格式

一行，一个整数k，表示最少引燃的炸弹数。

样例

样例输入


			
3 2

0 0

0 2

3 2

样例输出


			
2
