题目描述

n列火车，每条有l节车厢。每节车厢有一种颜色（用小写字母表示）。有m次车厢交换操作。求：对于每列火车，在交换车厢的某个时刻，与其颜色完全相同的火车最多有多少。

输入格式

n l m (2 ≤ n ≤ 1000, 1 ≤ l ≤ 100, 0 ≤ m ≤ 100000) n行字符串，长度为l m行，每行4个数a b c d,a车的第b个字符与c车第d个字符交换。

输出格式

n个数，在交换车厢的某个时刻，与该车颜色完全相同的火车最多数目。

样例

样例输入


			
5 6 7

ababbd

abbbbd

aaabad

caabbd

cabaad

2 3 5 4

5 3 5 5

3 5 2 2

1 2 4 3

2 2 5 1

1 1 3 3

4 1 5 6

样例输出


			
3

3

3

2

3
