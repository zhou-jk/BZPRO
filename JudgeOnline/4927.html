Problem 4927. -- 第一题

4927: 第一题

Time Limit: 20 Sec  Memory Limit: 512 MB
Submit: 93  Solved: 43
[Submit][Status][Discuss]

Description

给定n根直的木棍，要从中选出6根木棍，满足：能用这6根木棍拼
出一个正方形。注意木棍不能弯折。问方案数。
正方形：四条边都相等、四个角都是直角的四边形。

Input

第一行一个整数n。
第二行包含n个整数ai，代表每根木棍的长度。
n ≤ 5000, 1 ≤ ai ≤ 10^7

Output

一行一个整数，代表方案数。

Sample Input

8
4 5 1 5 1 9 4 5

Sample Output

3
