#2647. [Neerc2011]Journey

内存限制：128 MiB 时间限制：10 Sec
提交 提交记录 讨论

题目描述

初始在(0, 0)

给你N个命令串 每个命令串包含若干命令

每个命令形式是 GO LEFT RIGHT Fk之一

GO 往前走1

LEFT 左转90度

RIGHT 右转90度

Fk 执行第k个命令串

f1: GO F2 GO F2 GO F2

f2: F3 F3 F3 F3

f3: GO LEFT

命令轨迹如图

输入格式

 
第一行 N
接下来N行 每行先一个整数Ci表示命令数 接下来Ci个命令

输出格式

输出曾经到达的最远的距离 如果到了无穷远 输出“Infinity”

样例

样例输入


			
Sample Input1

3

6 GO F2 GO F2 GO F2

4 F3 F3 F3 F3

2 GO LEFT



Sample Output1

5



Sample Input2

1

2 GO F1



Sample Output2



Infinity
