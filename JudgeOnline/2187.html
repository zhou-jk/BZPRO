题目描述

分数 给你4个正整数a，b，c，d，求一个最简分数 p / q满足 a / b < p / q < c / d，若有多组解，输出q最小的一组，若仍有多组解，输出p最小的一组。

输入格式

本题有多组数据，有若干行，每行4个数a，b，c，d。以文件的末尾作为结束。

输出格式

对于输入的每组数据输出一个最简分数p/q。

样例

样例输入


			
1 3 1 2

2 1 3 1

2 1 4 1

1000 1001 1001 1002

样例输出


			
2/5

5/2

3/1

2001/2003



数据约定

	对于100%的数据 1 < =  a，b，c，d < = 109

	对于100%的数据 1 < = 数据组数<=500

	数据保证至少存在一个最简分数符合条件。
