Problem 1427. -- Unequalled Consumption

1427: Unequalled Consumption

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 49  Solved: 17
[Submit][Status][Discuss]

Description

对于方程W1*T1+W2*T2...+Wn*Tn=S这个方程,设其有不少于P组解. 
现在给出P,请输出最小的S. 注意T1到Tn均为非负整数
这里最小的S要求S是正数，例如P=1的时候，最小的S不能是0。

Input

第一行给出数字N. 
第二行N个数,代表Wi 
第三行一个数字M,代表有M个询问 
第四行M个数,每个数字代表一个P.
N<=5 Wi<=10 询问数M<=10 P,S<=10^18

Output

M行,每行一个数S,对应于相应的P

Sample Input

3
1 1 2
3
2 3 6

Sample Output

1
2
3

HINT
