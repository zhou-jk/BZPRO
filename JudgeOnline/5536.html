Problem 5536. -- String

5536: String

Time Limit: 10 Sec  Memory Limit: 512 MB
Submit: 7  Solved: 3
[Submit][Status][Discuss]

Description

给定一个长度为n且仅包含小写英文字母的字符串S。
你有一个字符串T，初始为空串。
你可以进行n次操作，每次操作你可以在T的前端或末尾加入一个任意字母。记
第i次操作后T在S中的出现次数为fi，你需要最大化ans=∑ifi

Input

第一行一个正整数n，表示字符串S的长度。
第二行一个长度为n的字符串S。
1≤n≤2×10^5

Output

一行一个整数，表示ans的最大值

Sample Input

6
abcabc

Sample Output

9

HINT
