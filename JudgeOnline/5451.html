Problem 5451. -- 字符串

5451: 字符串

Time Limit: 3 Sec  Memory Limit: 256 MB
Submit: 55  Solved: 25
[Submit][Status][Discuss]

Description

给定正整数m以及n个01串s1~sn，你需要求出长度为2m的反对称的包含这n个01串作为子串的01串的个数。对998244353取模。
一个01串s是反对称的当且仅当它对于1<=i<=|s|都满足s[i]≠s[|s|-i+1]。

Input

第一行两个整数n,m。接下来n行每行一个字符串s1~sn。
n<=6，|si|<=100，m<=500。

Output

一行一个整数表示答案。

Sample Input

2 3
011
001

Sample Output

4

HINT
