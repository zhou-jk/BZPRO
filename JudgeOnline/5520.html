Problem 5520. -- Subsequences

5520: Subsequences

Time Limit: 1 Sec  Memory Limit: 512 MB
Submit: 3  Solved: 3
[Submit][Status][Discuss]

Description

给定n个小写字符串，考虑 n!种连接它们的顺序，
问有多少种连接顺序最后得到的字符串有偶数个本质不同的子序列。
n ≤ 20。
∑ len ≤ 100000。

Input

第一行给出数字N，接下来N行每行一个字符串

Output

如题

Sample Input

4
ab
abb
bc
cd

Sample Output

16

HINT
