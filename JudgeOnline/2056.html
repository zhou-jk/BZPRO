题目描述

输入格式

输入的第一行为一个整数t。 接下来t行，每行包含九个自然数。

输出格式

输出t行 每行一个整数，表示2^a+2^b+2^c+2^d+2^e+2^f+2^g+2^h+i。

样例

样例输入


			
1

21 30 0 0 0 0 0 0 2147483647

样例输出


			
3223322629

【数据规模】

40% t<=1000

100% t<=100000 a,b,c,d,e,f,g,h<=60 i<=9223372036854775808
