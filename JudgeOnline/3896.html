3896: 求和

Description

Input

第一行一个整数T。
接下来T个整数，表示A0，A1…A(T-1)

Output

输出T+1个整数，表示A0，A1…AT。

Sample Input

3
0 0 1

Sample Output

0 166666668 500000004 333333336

HINT

T<=2000,所有系数为小于1e+9+7的非负整数。
